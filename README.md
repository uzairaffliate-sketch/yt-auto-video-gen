# 🎬 YT Auto Video Gen

**Turn your script into a high‑quality video using unlimited free stock media — fully automated, cloud‑rendered, zero cost.**

> ✅ 100% free  
> ✅ No credit card  
> ✅ No local rendering — everything runs in GitHub Actions  
> ✅ Multiple aspect ratios & resolutions  
> ✅ AI voiceover (TTS) or your own audio  
> ✅ Random transitions  
> ✅ Smart media matching via sentence‑transformers  

---

## 🧠 How It Works

1. **You paste your script** into the web app.  
2. **The app triggers a GitHub Actions workflow** via a Vercel serverless function.  
3. **The workflow** breaks the script into scenes, searches **many free stock sites** for relevant images/videos, selects the best match using AI, generates or downloads your voiceover, and assembles the final video with transitions.  
4. **Download the finished video** from the provided link (artifact stays available for 7 days).
